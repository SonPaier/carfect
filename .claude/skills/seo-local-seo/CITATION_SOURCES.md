## Citation Audit Process

1. **Choose canonical NAP format** — exact Name, Address, Phone to use everywhere
2. **Audit Tier 1 sources first** — check all 10 essential citations for accuracy
3. **Fix inconsistencies** — update any incorrect NAP data
4. **Build missing citations** — create profiles on sources you're not listed on
5. **Submit to data aggregators** — for broader distribution
6. **Add industry-specific citations** — relevant to your business type
7. **Monitor regularly** — quarterly audit to catch new inconsistencies

### Priority order for building citations
1. Tier 1 Essential (all 10)
2. Data Aggregators (all 3)
3. Top 5 industry-specific sources
4. Tier 2 General directories
5. Remaining industry-specific sources
6. Social media profiles
