# seo-local-seo

Expert guide for local SEO optimization.

## Purpose
Teaches Claude how to optimize for local search: Google Business Profile, NAP consistency, local citations, local keywords, review management, local landing pages, local pack rankings, and multi-location SEO.

## Activation Triggers
This skill activates when the user mentions: Google Business Profile, GBP, local citations, NAP consistency, local keywords, near me, local pack, review management, multi-location SEO, local SEO, store locator, service area, Google Maps.
