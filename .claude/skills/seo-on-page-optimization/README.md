# seo-on-page-optimization

Expert guide for on-page SEO optimization.

## Purpose
Teaches Claude how to analyze and optimize on-page SEO elements: title tags, meta descriptions, headings, content, images, internal linking, URLs, Open Graph tags, and canonical tags.

## Activation Triggers
This skill activates when the user mentions: meta tags, title tags, headings, H1-H6, content optimization, image alt text, internal linking, URL structure, on-page SEO, Open Graph, canonical tags.

## Files
| File | Lines | Description |
|------|-------|-------------|
| SKILL.md | ~280 | Core on-page SEO rules and best practices |
| COMMON_MISTAKES.md | ~200 | 15 most frequent on-page mistakes with fixes |
| CHECKLISTS.md | ~150 | Step-by-step audit checklists with pass/fail criteria |
| EXAMPLES.md | ~200 | 4 before/after optimization examples |

## MCP Tools Used
- `analyze_page` — comprehensive on-page analysis
- `analyze_headings` — heading structure validation
- `analyze_images` — image SEO audit
- `analyze_internal_links` — link mapping and anchor analysis
- `generate_meta_suggestions` — meta tag improvement suggestions

## Evaluation Scenarios
1. Page missing meta description
2. Page with duplicate H1 tags
3. Page with images missing alt text
4. Site with orphan pages
