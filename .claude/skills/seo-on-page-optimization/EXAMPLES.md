## Example 1: Blog Post Optimization

### Before (Score: 35/100)
```html
<title>Our Blog</title>
<meta name="description" content="">
<h1>Welcome to Our Blog</h1>
<h3>How to Do SEO</h3>  <!-- skipped H2 -->
<p>SEO is important. You should do SEO. SEO helps your website. SEO SEO SEO.</p>
<img src="IMG_4521.jpg">  <!-- no alt -->
<a href="/guide">click here</a>
```

**Issues:** Generic title, missing meta description, keyword-stuffed content, skipped heading level, no image alt, non-descriptive anchor.

### After (Score: 92/100)
```html
<title>How to Do an SEO Audit in 2026: Complete Step-by-Step Guide | Brand</title>
<meta name="description" content="Learn how to perform a comprehensive SEO audit in 2026. This step-by-step guide covers technical SEO, on-page optimization, and content analysis. Get started now.">
<link rel="canonical" href="https://example.com/seo-audit-guide">
<meta property="og:title" content="How to Do an SEO Audit in 2026">
<meta property="og:description" content="Complete step-by-step SEO audit guide covering technical, on-page, and content analysis.">
<meta property="og:image" content="https://example.com/images/seo-audit-guide-og.jpg">

<h1>How to Do an SEO Audit in 2026: Complete Step-by-Step Guide</h1>
<h2>What Is an SEO Audit?</h2>
```

## Key Takeaways

1. **Specific > Generic** — "Emergency Plumber in Austin TX" beats "Plumber"
2. **One clear H1** that matches what the user searched for
3. **Descriptive anchors** — "our SEO services" not "click here"
4. **Every image has purpose** — descriptive alt, optimized format and size
5. **Content depth** — enough content to demonstrate expertise
6. **Meta tags sell the click** — write descriptions that make users want to click
7. **Clean URLs** — keyword-rich, human-readable
